Hit

GitHub Pages 可以使用自己的域名，但是需要的 Repo 中跟目录下有一个 CNAME 文件包含你的域名
而 Hexo 每次使用 $ hexo d发布都会对分支进行修改,而手动建立的 CNAME 文件就会被删除。

解决办法为在 Hexo source目录下添加 CNAME 文件,这样 Hexo 自己会 push 这个文件到分支上。
